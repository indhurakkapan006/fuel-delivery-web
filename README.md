# ⛽ Fuel Buying Website

A full-stack web application that allows users to place fuel orders, track transactions, and manage their profiles through a responsive dashboard.

---

## 🚀 Features

- Fuel order placement
- Order tracking system
- User profile management
- Transaction history dashboard
- Responsive design
- Secure data handling

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3

### Backend
- Node.js
- Express.js
- REST APIs

### Database
- MySQL

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
git clone https://github.com/indhurakkapan006/fuel-delivery-web.git

### 2️⃣ Navigate to project folder
cd fuel-delivery-web

### 3️⃣ Install dependencies
npm install

### 4️⃣ Run the application
npm start
